Font Style

CSS dapat mengatur style dari teks yang ada pada halaman web yang dibuat. Terdapat beberapa property yang dapat digunakan untuk mengatur style teks yaitu sebagai berikut :

  • text-align : untuk mengatur perataan teks, value yang digunakan yaitu left, center dan right.
  • color : untuk memberikan warna pada teks.
  • letter-spacing : untuk mengatur jarak antar karakter.
  • word-spacing : untuk mengatur jarak antar kata.
